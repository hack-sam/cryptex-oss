# Auth v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace `ownerId: 'local'` with real Supabase-backed user accounts; introduce Guest / Free / Paid tiers with Godmode as the paid differentiator; preserve BYOK zero-knowledge posture via client-side passphrase crypto.

**Architecture:** Supabase handles identity (Google + GitHub OAuth), Postgres, Storage, Realtime, and Edge Functions. Dexie stays the authoritative local store; a sync engine pushes to Supabase in the background (last-writer-wins via `updatedAt`). BYOK keys encrypted client-side with PBKDF2+AES-GCM from a user passphrase so the server never sees plaintext. Every commit is gated behind `VITE_AUTH_ENABLED` so intermediate ships don't regress the current local-only experience.

**Tech Stack:** Supabase (`@supabase/supabase-js@^2.47.0`), Stripe (`stripe@^17.0.0` in Edge Functions only), WebCrypto (no third-party crypto libs), existing Svelte 5 + SvelteKit 2 + Dexie 4 stack, Vitest, Playwright for E2E.

**Spec:** `docs/superpowers/specs/2026-04-19-auth-v2-design.md`

**Parent commit:** `87e3cfb`

**Commit count:** 12 atomic commits. Each commit is shippable — build passes, 212+ existing tests pass, zero behavior change while `VITE_AUTH_ENABLED` is unset.

/** Null when VITE_AUTH_ENABLED is off. Callers MUST null-check before use. */
export const supabase: SupabaseClient | null = (() => {
  if (!featureFlags.authEnabled) return null;
  const url = import.meta.env.PUBLIC_SUPABASE_URL;
  const key = import.meta.env.PUBLIC_SUPABASE_ANON_KEY;
  if (!url || !key) {

    return null;
  }
  return createClient(url, key, {
    auth: { persistSession: true, autoRefreshToken: true, detectSessionInUrl: true }
  });
})();

create table profiles (
  id uuid primary key references auth.users on delete cascade,
  email text not null,
  plan text not null default 'free' check (plan in ('free', 'paid', 'grace')),
  stripe_customer_id text,
  grace_until timestamptz,

);

-- Auto-create a profile row when a new auth.user is created.
create or replace function public.handle_new_user()
returns trigger as $$
begin
  insert into public.profiles (id, email) values (new.id, new.email);

end;

```ts
import { supabase } from './supabase';
import { featureFlags } from '$lib/config/featureFlags';
import type { Session, User } from '@supabase/supabase-js';

export type CurrentUser = {
  id: string;
  email?: string;
  name?: string;
  avatarUrl?: string;
  plan: 'free' | 'paid' | 'grace';
};

// --- state ---------------------------------------------------------------
let _current = $state<CurrentUser | null>(
  featureFlags.authEnabled ? null : { id: 'local', plan: 'free' }
);
let _vaultKey = $state<CryptoKey | null>(null);
let _session = $state<Session | null>(null);

// Initialize Supabase session watcher once.
if (featureFlags.authEnabled && supabase) {
  supabase.auth.getSession().then(({ data }) => {
    _session = data.session;
    _current = shapeFromSession(data.session);
  });
  supabase.auth.onAuthStateChange((_event, session) => {
    _session = session;
    _current = shapeFromSession(session);
    if (!session) _vaultKey = null;   // sign-out locks vault
  });
}

function shapeFromSession(s: Session | null): CurrentUser | null {
  if (!s) return null;
  const u = s.user;
  const meta = (u.user_metadata ?? {}) as Record<string, unknown>;
  const plan = ((s.user.app_metadata as { plan?: string })?.plan ?? 'free') as CurrentUser['plan'];
  return {
    id: u.id,
    email: u.email,
    name: (meta.full_name as string | undefined) ?? (meta.name as string | undefined),
    avatarUrl: meta.avatar_url as string | undefined,
    plan
  };
}

// --- public API ----------------------------------------------------------
export const session = {
  get current() { return _current; },
  get isSignedIn() { return _current !== null && _current.id !== 'local'; },
  get vaultUnlocked() { return _vaultKey !== null; },
  get supabaseSession() { return _session; },

  async signInWithGoogle(): Promise<void> {
    if (!supabase) throw new Error('Auth not enabled');
    const { error } = await supabase.auth.signInWithOAuth({
      provider: 'google',
      options: { redirectTo: `${window.location.origin}/auth/callback` }
    });
    if (error) throw error;
  },

  async signInWithGitHub(): Promise<void> {
    if (!supabase) throw new Error('Auth not enabled');
    const { error } = await supabase.auth.signInWithOAuth({
      provider: 'github',
      options: { redirectTo: `${window.location.origin}/auth/callback` }
    });
    if (error) throw error;
  },

  async signOut(): Promise<void> {
    if (!supabase) return;
    await supabase.auth.signOut();
    _vaultKey = null;
  },

  // Vault lifecycle — stubs filled in C5
  /** @internal Set by key-vault.ts after successful unlock. */
  _setVaultKey(k: CryptoKey | null) { _vaultKey = k; },
  /** @internal Read by key-vault.ts. */
  _getVaultKey(): CryptoKey | null { return _vaultKey; }
};

/** Compatibility helper: every component calling `currentUser.id` today
 *  gets the same string ('local' when auth off, uuid when signed in). */
export function currentOwnerId(): string {
  return _current?.id ?? 'local';
}
```

import { supabase } from './supabase';
import { session } from './session.svelte';

const PBKDF2_ITERATIONS = 600_000;
const PBKDF2_HASH = 'SHA-256';
const AES_KEY_LENGTH = 256;
const SALT_BYTES = 32;
const IV_BYTES = 12;

// --- pure crypto helpers (unit-testable) ---------------------------------

export function generateSalt(): Uint8Array {
  return crypto.getRandomValues(new Uint8Array(SALT_BYTES));
}

export function generateIv(): Uint8Array {
  return crypto.getRandomValues(new Uint8Array(IV_BYTES));
}

export async function deriveKey(passphrase: string, salt: Uint8Array): Promise<CryptoKey> {
  const mat = await crypto.subtle.importKey(
    'raw',
    new TextEncoder().encode(passphrase),
    'PBKDF2',
    false,
    ['deriveKey']
  );
  return crypto.subtle.deriveKey(
    { name: 'PBKDF2', salt, iterations: PBKDF2_ITERATIONS, hash: PBKDF2_HASH },
    mat,
    { name: 'AES-GCM', length: AES_KEY_LENGTH },
    false,
    ['encrypt', 'decrypt']
  );
}

export async function encryptKey(plaintext: string, key: CryptoKey, iv: Uint8Array): Promise<ArrayBuffer> {
  return crypto.subtle.encrypt(
    { name: 'AES-GCM', iv },
    key,
    new TextEncoder().encode(plaintext)
  );
}

export async function decryptKey(ciphertext: ArrayBuffer, key: CryptoKey, iv: Uint8Array): Promise<string> {
  const plain = await crypto.subtle.decrypt({ name: 'AES-GCM', iv }, key, ciphertext);
  return new TextDecoder().decode(plain);
}

// --- high-level API --------------------------------------------------------

export type VaultKeyRow = {
  id: string;
  provider_id: string;
  instance_id: string | null;
  label: string | null;
  ciphertext: ArrayBuffer;
  iv: Uint8Array;
  salt: Uint8Array;
  kdf_iterations: number;
};

/** Attempt to unlock the vault with the given passphrase.
 *  Fetches all byok_keys rows and tries to decrypt the first.
 *  Returns true on success (derived key cached in session). */
export async function unlockVault(passphrase: string): Promise<boolean> {
  if (!supabase) return false;
  const { data, error } = await supabase.from('byok_keys').select('*').limit(1);
  if (error || !data || data.length === 0) {
    // No keys stored yet — treat this passphrase as the new master.
    // First encrypt will use it; cache nothing until then.
    return true;
  }
  const row = data[0] as unknown as { ciphertext: Uint8Array; iv: Uint8Array; salt: Uint8Array };
  try {
    const k = await deriveKey(passphrase, row.salt);
    await decryptKey(row.ciphertext.buffer, k, row.iv);
    session._setVaultKey(k);
    return true;
  } catch {
    return false;
  }
}

export function lockVault(): void {
  session._setVaultKey(null);
}

export async function storeBYOKKey(
  providerId: string,
  instanceId: string | null,
  apiKey: string,
  label: string | null,
  passphrase: string
): Promise<void> {
  if (!supabase) throw new Error('Auth not enabled');
  const salt = generateSalt();
  const iv = generateIv();
  const key = await deriveKey(passphrase, salt);
  const plaintext = JSON.stringify({ providerId, apiKey });
  const ciphertext = await encryptKey(plaintext, key, iv);

  const { error } = await supabase.from('byok_keys').upsert({
    owner_id: (session.current as { id: string }).id,
    provider_id: providerId,
    instance_id: instanceId,
    label,
    ciphertext: new Uint8Array(ciphertext),
    iv,
    salt,
    kdf_iterations: PBKDF2_ITERATIONS,
    kdf_hash: PBKDF2_HASH
  });
  if (error) throw error;
  session._setVaultKey(key);
}

export async function rotatePassphrase(oldP: string, newP: string): Promise<void> {
  if (!supabase) throw new Error('Auth not enabled');
  const { data, error } = await supabase.from('byok_keys').select('*');
  if (error || !data) throw error ?? new Error('No vault');

  const rows = data as unknown as Array<{ id: string; ciphertext: Uint8Array; iv: Uint8Array; salt: Uint8Array }>;

  // Decrypt all with old passphrase (one salt per row — but they share
  // session-level semantics; use each row's stored salt).
  const decrypted: Array<{ id: string; plaintext: string }> = [];
  for (const r of rows) {
    const oldKey = await deriveKey(oldP, r.salt);
    const pt = await decryptKey(r.ciphertext.buffer, oldKey, r.iv);
    decrypted.push({ id: r.id, plaintext: pt });
  }

  // Re-encrypt with new passphrase + fresh salt+iv per row.
  for (const d of decrypted) {
    const salt = generateSalt();
    const iv = generateIv();
    const newKey = await deriveKey(newP, salt);
    const ct = await encryptKey(d.plaintext, newKey, iv);
    const { error: updErr } = await supabase
      .from('byok_keys')
      .update({ ciphertext: new Uint8Array(ct), iv, salt })
      .eq('id', d.id);
    if (updErr) throw updErr;
  }

  // Cache the new key based on one of the rotated rows.
  if (rows.length > 0) {
    const lastSalt = (await supabase.from('byok_keys').select('salt').limit(1)).data?.[0]?.salt;
    if (lastSalt) {
      const k = await deriveKey(newP, lastSalt as unknown as Uint8Array);
      session._setVaultKey(k);
    }
  }
}

// In-memory rate limiter scoped to a single function instance.
// For production, Supabase recommends upstash/redis; this is the v1 floor.
const buckets = new Map<string, number[]>();

export function rateLimit(key: string, max: number, windowMs: number): boolean {
  const now = Date.now();
  const stamps = buckets.get(key)?.filter((t) => now - t < windowMs) ?? [];
  if (stamps.length >= max) return false;
  stamps.push(now);
  buckets.set(key, stamps);
  return true;
}

Deno.serve(async (req) => {
  if (req.method === 'OPTIONS') return new Response('ok', { headers: corsHeaders });
  const sig = req.headers.get('stripe-signature');
  if (!sig) return new Response('Missing stripe-signature', { status: 400, headers: corsHeaders });
  const secret = Deno.env.get('STRIPE_WEBHOOK_SECRET');
  if (!secret) return new Response('Webhook not configured', { status: 500, headers: corsHeaders });
  const body = await req.text();
  // HMAC verification — C11 fills in actual Stripe event handling
  const encoder = new TextEncoder();
  const key = await crypto.subtle.importKey('raw', encoder.encode(secret), { name: 'HMAC', hash: 'SHA-256' }, false, ['verify']);
  // Parse "t=...,v1=..." from signature header
  const parts = Object.fromEntries(sig.split(',').map(p => p.split('=')));
  if (!parts.t || !parts.v1) return new Response('Bad signature format', { status: 400, headers: corsHeaders });
  const signed = `${parts.t}.${body}`;
  const expected = hexFromBase64(parts.v1);
  const ok = await crypto.subtle.verify('HMAC', key, expected, encoder.encode(signed));
  if (!ok) return new Response('Signature mismatch', { status: 401, headers: corsHeaders });
  return new Response('ok', { status: 200, headers: corsHeaders });
});

function hexFromBase64(b64: string): Uint8Array {
  // Stripe sigs come as hex; decode accordingly
  const bytes = new Uint8Array(b64.length / 2);
  for (let i = 0; i < bytes.length; i++) bytes[i] = parseInt(b64.substr(i * 2, 2), 16);
  return bytes;
}

`app/src/lib/sync/queue.ts`:
```ts
import { db } from '$lib/chat/db';
import type { SyncQueueRow, SyncOp } from '$lib/chat/db';

const MAX_QUEUE = 10_000;

export const syncQueue = {
  async enqueue(table: SyncQueueRow['table'], rowId: string, op: SyncOp): Promise<void> {
    await db.syncQueue.add({ table, rowId, op, enqueuedAt: Date.now() } as SyncQueueRow);
    if (await db.syncQueue.count() > MAX_QUEUE) await this.coalesce();
  },

  async drain(max: number): Promise<SyncQueueRow[]> {
    return db.syncQueue.orderBy('seq').limit(max).toArray();
  },

  async ack(seqs: number[]): Promise<void> {
    await db.syncQueue.bulkDelete(seqs);
  },

  /** Collapse duplicate (table,rowId) entries, keeping the latest op.
   *  Delete always wins over upsert regardless of order. */
  async coalesce(): Promise<number> {
    const all = await db.syncQueue.orderBy('seq').toArray();
    const kept = new Map<string, SyncQueueRow>();
    for (const row of all) {
      const key = `${row.table}:${row.rowId}`;
      const prev = kept.get(key);
      if (!prev) { kept.set(key, row); continue; }
      if (row.op === 'delete') kept.set(key, row);
      else if (prev.op !== 'delete') kept.set(key, row);
      // else: prev is delete, keep it
    }
    const keptSeqs = new Set([...kept.values()].map(r => r.seq!));
    const toDelete = all.filter(r => !keptSeqs.has(r.seq!)).map(r => r.seq!);
    await db.syncQueue.bulkDelete(toDelete);
    return toDelete.length;
  }
};
```

`app/src/lib/sync/engine.ts`:
```ts
import { supabase } from '$lib/auth/supabase';
import { session } from '$lib/auth/session.svelte';
import { syncQueue } from './queue';
import { db } from '$lib/chat/db';

const TICK_MS = 2000;
const BATCH_SIZE = 100;
const BACKOFF_MS = [1000, 2000, 4000, 8000, 30000];

let running = false;
let tickTimer: number | null = null;
let backoffIdx = 0;

export const syncEngine = {
  start(): void {
    if (running || !supabase || !session.isSignedIn) return;
    running = true;
    backoffIdx = 0;
    this.tick();
  },

  stop(): void {
    running = false;
    if (tickTimer) { clearTimeout(tickTimer); tickTimer = null; }
  },

  async tick(): Promise<void> {
    if (!running || !supabase) return;
    try {
      const batch = await syncQueue.drain(BATCH_SIZE);
      if (batch.length > 0) {
        // Group by (table, op) then push in bulk
        const grouped = new Map<string, typeof batch>();
        for (const row of batch) {
          const k = `${row.table}:${row.op}`;
          (grouped.get(k) ?? grouped.set(k, []).get(k)!)!.push(row);
        }
        for (const [k, rows] of grouped) {
          const [table, op] = k.split(':') as [string, 'upsert' | 'delete'];
          if (op === 'delete') {
            await supabase.from(table).update({ tombstoned: true }).in('id', rows.map(r => r.rowId));
          } else {
            // Fetch full row from Dexie, upsert to Supabase
            const localRows = await Promise.all(rows.map(r => this.fetchLocal(table, r.rowId)));
            const upsertable = localRows.filter(Boolean);
            if (upsertable.length > 0) await supabase.from(table).upsert(upsertable, { onConflict: 'id' });
          }
        }
        await syncQueue.ack(batch.map(b => b.seq!));
      }
      backoffIdx = 0; // reset backoff on success
    } catch (err) {
      console.error('[sync] tick failed:', err);
      backoffIdx = Math.min(backoffIdx + 1, BACKOFF_MS.length - 1);
    }
    const delay = backoffIdx === 0 ? TICK_MS : BACKOFF_MS[backoffIdx];
    tickTimer = setTimeout(() => this.tick(), delay) as unknown as number;
  },

  async fetchLocal(table: string, rowId: string): Promise<Record<string, unknown> | null> {
    // Map table name -> Dexie collection
    const m: Record<string, any> = {
      chats: db.chats,
      messages: db.messages,
      attachments: db.attachments,
      attack_chain_runs: db.attackChainRuns
    };
    const coll = m[table];
    if (!coll) return null;
    return coll.get(rowId) ?? null;
  }
};

import { db } from '$lib/chat/db';
import { syncQueue } from './queue';

export type ClaimResult = {
  chats: number;
  messages: number;
  attachments: number;
  attackChainRuns: number;
};

/** Rewrite ownerId='local' to the real uuid across every synced table.
 *  Atomic via single Dexie transaction. Idempotent — running twice is
 *  a no-op because after the first pass, no 'local' rows remain. */
export async function claimLocalData(newOwnerId: string): Promise<ClaimResult> {
  const result: ClaimResult = { chats: 0, messages: 0, attachments: 0, attackChainRuns: 0 };

  await db.transaction(
    'rw',
    db.chats, db.messages, db.attachments, db.attackChainRuns,
    async () => {
      result.chats           = await db.chats          .where('ownerId').equals('local').modify({ ownerId: newOwnerId, updatedAt: Date.now() });
      result.messages        = await db.messages       .where('ownerId').equals('local').modify({ ownerId: newOwnerId, updatedAt: Date.now() });
      result.attachments     = await db.attachments    .where('ownerId').equals('local').modify({ ownerId: newOwnerId, updatedAt: Date.now() });
      result.attackChainRuns = await db.attackChainRuns.where('ownerId').equals('local').modify({ ownerId: newOwnerId, updatedAt: Date.now() });
    }
  );

  // Enqueue every touched row for sync push.
  if (result.chats > 0) {
    const chats = await db.chats.where('ownerId').equals(newOwnerId).toArray();
    for (const c of chats) await syncQueue.enqueue('chats', c.id, 'upsert');
  }
  // (Same pattern for messages, attachments, attack_chain_runs — omitted for brevity; replicate.)

  return result;
}

```ts
import { corsHeaders } from '../_shared/cors.ts';
import Stripe from 'https://esm.sh/stripe@17.0.0?target=deno&no-check';
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2';

const stripe = new Stripe(Deno.env.get('STRIPE_SECRET_KEY')!, { apiVersion: '2024-12-18.acacia' });

Deno.serve(async (req) => {
  if (req.method === 'OPTIONS') return new Response('ok', { headers: corsHeaders });
  const sig = req.headers.get('stripe-signature');
  if (!sig) return new Response('Missing stripe-signature', { status: 400 });
  const body = await req.text();

  let event: Stripe.Event;
  try {
    event = await stripe.webhooks.constructEventAsync(body, sig, Deno.env.get('STRIPE_WEBHOOK_SECRET')!);
  } catch (err) {
    return new Response(`Signature verification failed: ${(err as Error).message}`, { status: 401 });
  }

  const service = createClient(Deno.env.get('SUPABASE_URL')!, Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!);

  switch (event.type) {
    case 'checkout.session.completed': {
      const s = event.data.object as Stripe.Checkout.Session;
      const customerId = typeof s.customer === 'string' ? s.customer : s.customer?.id;
      if (customerId) {
        await service.from('profiles').update({ plan: 'paid' }).eq('stripe_customer_id', customerId);
      }
      break;
    }
    case 'customer.subscription.deleted': {
      const sub = event.data.object as Stripe.Subscription;
      const customerId = typeof sub.customer === 'string' ? sub.customer : sub.customer.id;
      await service.from('profiles').update({ plan: 'free', grace_until: null }).eq('stripe_customer_id', customerId);
      break;
    }
    case 'customer.subscription.updated': {
      const sub = event.data.object as Stripe.Subscription;
      const customerId = typeof sub.customer === 'string' ? sub.customer : sub.customer.id;
      if (sub.status === 'past_due') {
        const grace = new Date(Date.now() + 7 * 86400_000).toISOString();
        await service.from('profiles').update({ plan: 'grace', grace_until: grace }).eq('stripe_customer_id', customerId);
      } else if (sub.status === 'active') {
        await service.from('profiles').update({ plan: 'paid', grace_until: null }).eq('stripe_customer_id', customerId);
      }
      break;
    }
    default:
      // Unhandled — ignore but return 200
      break;
  }

  return new Response(JSON.stringify({ received: true }), { headers: { 'Content-Type': 'application/json' }});
});
```

# Auth v2 Deployment Runbook

## 1. Create Supabase project
- https://supabase.com/dashboard/new
- Region: pick closest to majority of users
- Set org-level MFA on your own Supabase account

## 2. Apply schema
```bash
npx supabase link --project-ref <your-ref>
npx supabase db push
```

## 3. Configure OAuth providers
Dashboard → Authentication → Providers:
- Enable Google (client ID + secret from Google Cloud Console)
- Enable GitHub (client ID + secret from https://github.com/settings/applications/new)
- Set redirect URL: https://<your-project>.supabase.co/auth/v1/callback
- Add your app origin(s) to "Allowed URLs"

## 4. Configure Stripe
- Create Product + Price (monthly recurring)
- Create webhook endpoint pointing to:
  https://<your-project>.supabase.co/functions/v1/stripe-webhook
  Events: checkout.session.completed, customer.subscription.{created,updated,deleted}
- Capture `STRIPE_WEBHOOK_SECRET` (whsec_...)

## 5. Set Edge Function secrets
```bash
supabase secrets set \
  STRIPE_SECRET_KEY=sk_live_... \
  STRIPE_WEBHOOK_SECRET=whsec_... \
  STRIPE_PRICE_ID=price_... \
  APP_ORIGIN=https://cryptex.example.com
```

## 6. Deploy Edge Functions
```bash
npx supabase functions deploy create-checkout-session
npx supabase functions deploy stripe-webhook
npx supabase functions deploy create-billing-portal-session
npx supabase functions deploy delete-account
npx supabase functions deploy export-account
npx supabase functions deploy godmode-prompt
```

## 7. Flip the app flag
In your host env (Cloudflare Pages / Vercel / Dokploy):
```
VITE_AUTH_ENABLED=true
PUBLIC_SUPABASE_URL=https://<your-project>.supabase.co
PUBLIC_SUPABASE_ANON_KEY=<anon key from Supabase Dashboard>
```

Redeploy. Sign-in wall appears on /chat and /dataset.

## 8. Smoke test
1. Sign up with a personal Google account
2. Create a chat, send a message
3. Sign out
4. Sign in from a second device — same chat appears
5. Click Godmode in chat → UpgradeModal appears
6. Stripe test mode: complete checkout → plan flips to paid → Godmode unlocks

## 9. Rollback
Set `VITE_AUTH_ENABLED=false` in host env and redeploy. App reverts to local-only mode. Existing Supabase data untouched.

## 10. Monitoring
- Supabase Dashboard → Logs → Edge Functions (webhook failures)
- Stripe Dashboard → Webhooks (retry failures)
- Supabase Dashboard → Database → Logs (RLS policy violations)
```
